## 제품 소개

APIS 패브릭마커는 **섬유 전용으로 개발**된 수성 마커입니다.
다림질 열처리 후 세탁해도 색이 유지되는 특수 잉크로 창작 활동을 오래 보존합니다.

## 핵심 특징

- **섬유 전용 잉크**: 천·면·폴리에스터 섬유에 최적화
- **세탁 내성**: 열처리(다림질) 후 일반 세탁에도 색상 유지
- **16색 풀세트**: 창작에 필요한 모든 색상 포함
- **무독성 수성 포뮬러**: 어린이도 안전하게 사용 가능

## 사용 방법

1. 원하는 디자인을 섬유에 직접 그리기
2. 24시간 건조
3. 다림질로 열처리 (뒤집어서 면 위에 다림질)
4. 이후 세탁 가능

## OEM 제조 안내

- 색상 조합 커스터마이징
- 아동용 안전 인증 추가 가능
- MOQ: 500세트부터
