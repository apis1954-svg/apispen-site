## 제품 소개

APIS 마커 세트 베이직은 **가정·사무·학교에서 가장 많이 사용하는 마커 7종**을 한 번에 구성했습니다.
유성마커, 네임펜, 보드마커, 형광펜을 모두 포함한 올인원 세트입니다.

## 구성 내용

1. **유성마커 블랙** — 범용 마킹, 방수
2. **유성마커 레드** — 중요 표시
3. **네임펜 블랙** — 이름·라벨 쓰기
4. **보드마커 블랙** — 화이트보드 전용
5. **보드마커 블루** — 화이트보드 보조색
6. **형광펜 노랑** — 중요 내용 하이라이팅
7. **형광펜 핑크** — 보조 하이라이팅
